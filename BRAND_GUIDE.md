## Typography

### Font Family

#### Headings: Poppins

- **Weight**: Bold (600), Extra-Bold (700), Black (800)
- **Purpose**: Eye-catching, modern, energetic
- **Usage**: H1-H6, buttons, callouts

#### Body: Inter

- **Weight**: Regular (400), Medium (500), Semi-Bold (600)
- **Purpose**: Clean, readable, modern
- **Usage**: Body text, labels, descriptions

### Font Hierarchy

| Level      | Font    | Size            | Weight | Usage              |
| ---------- | ------- | --------------- | ------ | ------------------ |
| H1         | Poppins | 3.5rem (56px)   | 800    | Page headline      |
| H2         | Poppins | 2.5rem (40px)   | 700    | Section headers    |
| H3         | Poppins | 1.5rem (24px)   | 600    | Card headers       |
| Body Large | Inter   | 1.25rem (20px)  | 400    | Large body text    |
| Body       | Inter   | 1rem (16px)     | 400    | Standard body text |
| Body Small | Inter   | 0.875rem (14px) | 400    | Secondary text     |
| Label      | Inter   | 0.75rem (12px)  | 600    | Labels, tags       |

### Typography Pairings

- **Headlines**: Poppins Bold/Extra-Bold
- **Body**: Inter Regular
- **Emphasis**: Inter Semi-Bold
- **Metadata**: Inter Regular, lighter color

### Text Styling

- **Links**: Blue (#0066FF), underline on hover
- **Active**: Blue with bold weight
- **Hover**: Darker shade, slight weight increase
- **Disabled**: Light gray, reduced opacity

## Voice & Tone

### Brand Voice Characteristics

- **Motivational**: Inspiring confidence and action
- **Pragmatic**: Realistic, grounded, practical
- **Empathetic**: Understanding and supportive
- **Confident**: Knowledge-based, authoritative

### Tone by Context

#### Marketing Copy

- Benefit-focused ("What you'll gain")
- Action-oriented ("Get started today")
- Inspiring ("Transform your health")

#### Educational Content

- Clear and concise
- Evidence-based
- Encouraging without hype

#### Support & Help

- Friendly and approachable
- Jargon-free
- Solution-focused

#### Error Messages

- Clear, not blaming
- Helpful suggestions
- Path to resolution

### Writing Guidelines

#### Dos

✓ Use active voice ("You'll achieve" not "Results can be achieved")
✓ Use second person ("Your personalized plan")
✓ Be specific ("Lose 12 pounds" not "Lose weight")
✓ Use benefit-driven language
✓ Keep sentences short and scannable

#### Don'ts

✗ Use passive voice or jargon
✗ Make unrealistic claims
✗ Use complex medical terminology
✗ Be condescending
✗ Use excessive punctuation

### Example Copy

**Good**: "Get your personalized fitness plan in 5 minutes."
**Bad**: "A personalized fitness plan can be obtained."

**Good**: "Wellness that works with your life, not against it."
**Bad**: "Our app is comprehensive."

## Design Patterns

### Spacing System

```
xs: 0.5rem (8px)
sm: 1rem (16px)
md: 1.5rem (24px)
lg: 2rem (32px)
xl: 3rem (48px)
2xl: 4rem (64px)
```

### Border Radius

- **Small**: 6-8px (buttons, small elements)
- **Medium**: 12px (cards, moderate elements)
- **Large**: 16px (major cards, containers)
- **Full**: 50% (avatars, circles)

### Shadows

- **Subtle**: `0 4px 16px rgba(0, 0, 0, 0.08)`
- **Medium**: `0 8px 32px rgba(0, 0, 0, 0.12)`
- **Strong**: `0 20px 60px rgba(0, 0, 0, 0.3)`

### Transitions

- **Default**: 0.3s ease
- **Quick**: 0.15s ease
- **Smooth**: 0.5s ease-out

## Brand Do's and Don'ts

### Do

✓ Use brand colors consistently
✓ Maintain proper spacing and hierarchy
✓ Use Poppins for headings, Inter for body
✓ Keep imagery modern and diverse
✓ Write in active, benefit-driven voice
✓ Stay focused on user benefits
✓ Use gradients strategically

### Don't

✗ Change brand colors without approval
✗ Use other fonts without permission
✗ Mix different design styles
✗ Use outdated or generic imagery
✗ Oversell or make false claims
✗ Ignore accessibility standards
✗ Crowd layouts with too much information
